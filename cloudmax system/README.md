## 🎯 What is CLOUDMAX NEXUS?

## 🚀 Quick Start (5 Minutes)

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

```bash
# 1. Clone or create React app
npx create-react-app cloudmax-nexus
cd cloudmax-nexus

# 2. Install dependencies
npm install lucide-react axios socket.io-client

# 3. Configure Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# 4. Copy cloudmax-nexus.jsx to src/App.js
# (See QUICK_START_GUIDE.md for detailed steps)

# 5. Run locally
npm start

# 6. Test
# Login: admin@example.com / cloudmax123
```

### Deploy to Vercel (1 Click)

```bash
npm i -g vercel
vercel
# Follow prompts - live in 60 seconds!
```

**For detailed setup instructions, see [QUICK_START_GUIDE.md](./QUICK_START_GUIDE.md)**

## 🔌 API Integration Examples

### Call AI Agent
```javascript
const response = await fetch('/api/agents/execute', {
  method: 'POST',
  body: JSON.stringify({
    agent_id: 'CloudAnalyzer',
    action: 'analyze_infrastructure',
    parameters: { provider: 'aws', region: 'us-east-1' }
  })
});
const result = await response.json();
```

### Get Real-Time Metrics
```javascript
const socket = io('https://api.cloudmax-nexus.com');
socket.on('metrics-update', (metrics) => {
  console.log('Updated metrics:', metrics);
});
```

### DeepSeek Analysis
```javascript
const deepAnalysis = await fetch('/api/deepseek/analyze', {
  method: 'POST',
  body: JSON.stringify({
    query: 'Optimize our AWS infrastructure for cost and performance',
    depth: 'comprehensive'
  })
});
```
